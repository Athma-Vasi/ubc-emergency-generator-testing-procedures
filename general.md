# Emergency Generator Testing - Personal Shift Reference

> Unofficial personal reference based on Procedure 11 and site practice. Use
> approved UBC procedures, permits, and supervisory direction as the governing
> standard.

## 1) Test Intent

- Complete monthly functional testing for emergency power reliability.
- Validate generator, ATS operation, load transfer, and auxiliary equipment
  performance.
- Capture readings, abnormalities, and follow-up actions clearly.

## 2) Core Compliance Targets (Procedure 11 / CSA282-15)

- Simulate failure of normal electrical supply.
- Confirm battery charger current output increases during cranking.
- Run at >= 40% rated load for >= 60 minutes.
- Operate all associated ATS under load.
- Inspect applicable mechanical/electrical condition items.
- Verify auxiliary equipment operation.
- Record readings and correct/flag defects.

## 3) Pre-Test Setup Checklist

- [ ] Review schedule and work order details (Planon).

## 4) Test Execution Sequence

1. Start generator from transfer switch location (per site procedure).
2. Transfer ATS/load to emergency source as scheduled.
3. Confirm generator stabilizes (speed/voltage/frequency indications normal).
4. Run all associated ATS under load.
5. Continue run for at least 60 minutes.

## 5) Timed Readings and Checks

### At Start (0-5 min)

- [ ] Verify charger current response during crank.
- [ ] Observe for abnormal noise, vibration, alarms, smoke, or odor.
- [ ] Confirm life-safety/critical loads transferred as expected.

### Mid-Test (30 min)

- [ ] Record generator amperage.
- [ ] Record oil pressure.
- [ ] Record coolant temperature.
- [ ] Confirm stable operation and no active faults.

### During 60-minute Run

- [ ] Inspect brushes for sparking (if applicable).
- [ ] Inspect for motor bearing seal leakage.
- [ ] Check auxiliary equipment operation:
- [ ] Radiator shutter control.
- [ ] Coolant pumps.
- [ ] Fuel transfer pumps.
- [ ] Oil coolers.
- [ ] Engine room ventilation.
- [ ] Drain exhaust condensate trap (if required during/after run per site
      practice).

## 6) Fire Alarm Panel Handling

- [ ] Check fire alarm panel for trouble/supervisory conditions.
- [ ] Acknowledge and perform one reset attempt only.
- [ ] If reset fails, contact FLS for further investigation.

## 7) Re-Transfer and Shutdown

1. After >= 60 minutes, transfer load back to normal source.
2. Confirm ATS re-transfer complete and loads normalized.
3. Shut generator down per standard sequence.
4. Notify TELUS to release system from test.

## 8) Post-Test Documentation

- [ ] Log all readings and observations onsite.
- [ ] Record defects, alarms, unusual behavior, and temporary mitigations.
- [ ] Notify supervisor of issues requiring follow-up.
- [ ] Attach photos and timestamped notes to this file or linked records.
